# رادار الموديلات

لوحة تحكم لمتابعة مبيعات الموديلات يوميًا وأسبوعيًا عبر الفروع، ومعرفة أي الأصناف تبيع بشكل أفضل لطلبها أو نقلها بين الفروع، قبل أن تنفد.

## أهم الميزات

- **محرّك توصيات ذكي**: يحلّل كل موديل رائج في كل فرع، ويقارن نسبة البيع بالرصيد المتاح، ثم يقترح إمّا **نقل** الكمية الفائضة من فرع بطيء الحركة، أو **طلب** كمية جديدة من المورّد إذا لم يوجد فائض — مع شرح واضح بالعربية لسبب كل توصية.
- **مستكشف الموديلات**: بحث وفرز وتصفية عبر كل الموديلات، مع توزيع مرئي للمبيعات على الفروع وتفاصيل كاملة لكل موديل.
- **الاتجاهات الأسبوعية**: بمقارنة لقطتين (مثلاً هذا الأسبوع بالأسبوع الماضي) تحصل على سرعة البيع الفعلية لكل موديل والوقت المتوقع لنفاد المخزون.
- **خريطة حرارية** لأداء كل موديل عبر الفروع الخمسة بنظرة واحدة.
- **رفع بيانات دوري**: ارفع تقرير "Top Models" (بنفس تنسيق الأعمدة الحالي) يوميًا أو أسبوعيًا لبناء تاريخ يتيح رصد الاتجاهات.
- تخصيص أسماء الفروع وحساسية التوصيات من الإعدادات.

جميع البيانات تُخزَّن محليًا في متصفحك (IndexedDB) — لا يوجد خادم أو رفع لأي بيانات إلى الإنترنت.

## التشغيل محليًا

```bash
npm install
npm run dev
```
